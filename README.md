# Meme generator
### Описание
Создаёь мемы путём выбора изображения (из выпадающего списка) и ввода текста вверху и внизу выбранного изображения.

### Языки и техногии
Последние версии HTML и CSS, язык программирования JavaScript, API ImgFlip.
